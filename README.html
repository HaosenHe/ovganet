Overlapping Group Elastic Net Using the Orthogonalizing EM (OEM) Algorithm

Introduction

ovganet is an R package that fits overlapping group elastic nets using the latent group lasso approach (Obozinski et al., 2011). Our package is an extension of the oem package developed by Huling, J.D. and Chien, P., based on Xiong, S. et al., 2016. Certain parts of our code are adapted from grpregOverlap.

Author(s): Haosen He, Alison Morantz

Get Started

Lets first import the package:

library(ovganet)
## Loading required package: Matrix
## Loading required package: oem
## Loading required package: bigmemory

And a test data set:

data(mtcars)

Convert the data set to desired formats:

X <- as.matrix(mtcars[,-1])
y <- as.vector(mtcars$mpg)

Define an overlapping group (numbers represent column numbers in X)

group = list(c(1,2), c(2,3), c(3,4,5),c(4,5,6))

Perform overlapping group lasso:

fit <- ovganet(X = X, y = y, group = group, family ='gaussian')

Plot the path plot:

plot(fit)

Fit cross-validated model:

cvfit <- cv.ovganet(X = X, y = y, group = group, family ='gaussian')

Plot CV plot:

plot(cvfit)
